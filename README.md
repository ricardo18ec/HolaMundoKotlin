# Entrega #1 – Hola Mundo en Kotlin

## Propósito del repositorio

El propósito de este repositorio es introducir al lenguaje de programación Kotlin y poder
familiarizarse desde cero con sus conceptos básicos, como su objetivo principal, las formas en las que es utilizado y su sintaxis.

El proyecto incluye un programa sencillo que demuestra la correcta ejecución de código en Kotlin mediante el despliegue del mensaje “Hola Mundo” } y una multiplicación.
